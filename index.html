Exercise #1 (sugerencia: join())

El método join() en JavaScript se utiliza para unir todos los elementos de un arreglo en una sola cadena de texto, separándolos por un separador que tú elijas. Si no le pasas un separador, por defecto usará una coma (,).

Exercise #2 (sugerencia map())

El método map() en JavaScript es una función que se aplica a los arreglos (arrays) y sirve para crear un nuevo arreglo al transformar cada uno de los elementos del arreglo original. El map() no modifica el arreglo original, sino que devuelve un nuevo arreglo con los resultados de aplicar una función a cada elemento del original.

Exercise #3 (sugerencia reduce())

El método reduce() en JavaScript se utiliza para reducir (o "acumular") todos los elementos de un arreglo a un solo valor. A diferencia de otros métodos como map() o filter(), que devuelven nuevos arreglos, reduce() devuelve un único resultado, como un número, una cadena, o incluso un objeto.

Exercise #4 (sugerencia filter() e includes())

El método filter() en JavaScript se utiliza para crear un nuevo arreglo con aquellos elementos del arreglo original que cumplan una condición específica. Es decir, filtra los elementos del arreglo basado en una función que retorna true o false. Si la función devuelve true para un elemento, ese elemento se incluye en el nuevo arreglo; si devuelve false, se excluye.

Exercise #5

Un array es una estructura de datos o un tipo de objeto que se utiliza para almacenar múltiples valores en una sola variable. Un Array es una lista ordenada de elementos (como números, cadenas, objetos, etc.), donde cada elemento tiene una posición o índice.

OPTIONAL: Exercise # 6

l algoritmo de burbujas (o Bubble Sort en inglés) es un método sencillo para ordenar un arreglo (o lista) de elementos. Su funcionamiento se basa en la comparación de elementos adyacentes y la realización de intercambios para colocarlos en el orden correcto. A medida que avanza a través del arreglo, los elementos "más pesados" (en el caso de ordenar de menor a mayor) "flotan" hacia la parte superior, similar a burbujas en un líquido, de ahí su nombre.
